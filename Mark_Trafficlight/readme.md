# 简要使用说明

## 1.简介

该程序用于手动标定红绿灯位置并自动写入报文，最近一次更新在2022/7/14 

### 1.需要安装的库

python3

opencv-python



### 2.使用前的准备

1.给ubuntu安装python以及opencv-python并配置好环境变量

2.下载程序文件catch_the_photo.py

3.编辑python文件将其中的txt_addr变量更改为报文txt所在目录



### 3.具体操作步骤（前提:已完成所有准备工作）

1.cd到程序目录下打开终端并运行程序

2.出现窗口即可开始标定



### 4.标定的基本操作（注意:必须是英文输入法）

按下“n”（next）键切换到数字增大顺序的下一张图片

按下”l“（last）键回到上一张

当前图片下单击鼠标左键进行标定 (注意：单张图片只可标记两个点超过两个将重新从第一个点开始标定)

按下“c”（clean）键重新开始标定

按下esc键退出标定



### 5.标定具体操作

1.按照要求点击左上方向为第一个点，点击右下方为第二个点（点击完毕会显示两点的坐标与所成直线）

2.若标记无误可按下n键切换到下一张继续重复上述操作

3.若想要重新标记上一张可按下l键回到上一张

4.最后一张标定完毕后可以按下esc键退出或者n键使程序崩溃退出



### 6.输出

标记的所有的坐标内容将按照一定格式写回到对应行的报文txt文件
